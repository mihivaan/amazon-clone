# Amazon Homepage Clone

Frontend clone of Amazon's homepage using HTML, CSS, and JavaScript. Created to practice layout and styling.

## Features
- Static web layout
- Responsive design
- Clean navigation bar and footer

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Use
Open `index.html` in a web browser.
